Lotka-Volterra Predator-Prey Model Project

Project Overview

This was a group project in the class MATH 1270 (Ordinary Differential Equations) at the University of Pittsburgh. The paper and presentation describe and solve a specfic situation concerning the Lotka-Volterra Equations, better known as the Predator-Prey Equations. These equations are a common fundamental system of differential equations, used to describe the balance of populations between a prey and predator.

Objectives and Methodologies

The objective of this project was to describe the effects of an unknown parameter. As seen by the images to the left, setting the parameter to different values led to different phase portraits. Our goal was to find the bifurcation points, in which the phase portraits changed from one value to the next. To do this, we used basic tools like examining the nullclines of the system and finding the stability of equibibrium points to matehmatically decribe where the shifts occured. We also used MATLAB to produce figures to illustrate these changes and accurately argue where the bifurcation points were. Latex was used again for both the paper and presentation.
