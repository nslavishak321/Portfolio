Time-Series Forecasting Project

Project Overview

I developed a stock price prediction model using PyTorch and an LSTM (Long Short-Term Memory) neural network. The model forecasts the prices of three selected stocks one month into the future based on their historical closing prices. I preprocessed the time series data, normalized the values, and structured it into sequences suitable for LSTM input. After training the model, I evaluated its performance using metrics such as RMSE and visualized predicted vs. actual prices. This project demonstrates my ability to apply deep learning techniques to real-world financial forecasting tasks.

Objectives and Methodologies

The primary objective of this project was to predict stock prices one month into the future using historical data, leveraging the sequential modeling capabilities of LSTM networks. I began by collecting daily closing prices for three different stocks and performed data cleaning and normalization to ensure consistency and improve model performance. The time series data was then structured into rolling windows to create input-output sequences for supervised learning. I implemented an LSTM model in PyTorch, chosen for its effectiveness in capturing long-term dependencies in temporal data. Hyperparameters such as learning rate, number of layers, and hidden units were tuned to optimize model accuracy. The model was trained and validated using a sliding window approach, and its performance was assessed using Root Mean Squared Error (RMSE). This methodology allowed for dynamic learning from past trends and aimed to provide realistic forecasts useful for financial analysis and decision-making.
